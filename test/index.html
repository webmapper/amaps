Amaps demo

 The Amsterdam version of NL Maps uses the official Amsterdam topography and Amsterdam interface style.

 Above is a basic map, showing a marker to mark a certain location. The search box only shows adresses, and only within Amsterdam. Selecting an adress zooms to that location.

 We have two other examples with functionality for specific use cases, using bundled scripts to query certain API's when interacting with the map:

  • Point query example shows information about a selected coordinate.
  • Multi-feature select example shows information about one or more objects selected.

 For documentation on how to use these maps in your websites see Amaps applications on GitHub.
 Tip: download the source of this page, add <base href="https://map.data.amsterdam.nl/"> in the head-tag, and play around with it.

 For the source code and documentation, see GitHub.
